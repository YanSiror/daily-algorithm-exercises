Registration notes

If you wish to use Easy CHM after the evaluation period of 30 days, you need to purchase its license via ShareIT.

Upon receipt of your registration fee you will receive an email containing a registration key and a key file corresponding to a user name string which you have chosen. Please specify a valid email address when registering, and it will be used to send you registration key and a key file - please make sure your email address can receive email with attachment.

Registration email will also contain all necessary instructions, so just follow them. Below we provide a brief explanation of typical registration procedure, but instructions in the email are more up to date than this file and should have a higher precedence.

If you use Easy CHM, you will need to copy the registration key file to Easy CHM folder. By default it is "C:\Program Files\EasyCHM", but it can be changed by a user when installing Easy CHM. Then run Easy CHM and select "Enter registration code" from Register menu, enter your registration name and registration key, press button Ok. 

When registered correctly, the "Buy now" button and "Register" menu will disappear from the program's main screen, and then you can use the registered version of Easy CHM.

If the key file is archived, in a .zip file, please extract the key file from the archive before copying it.

If you do not get your registration key and a key file beyond 24 hours, please contact us at once. We will be happy to help you. If you can't get a reply from us, please try to contact us with a different email address (for example, gmail or Hotmail e-mail box) - maybe our registration email has been blocked by your email filters or email server because of the key file in the attachment.  

If you can't receive email with attachments, you can contact us and we'll provide a link for you to download the key file, or you can provide us a different email address to receive the registration email. 

Please send your further questions about sales and licensing to webmaster@zipghost.com or webmaster@etextwizard.com.
